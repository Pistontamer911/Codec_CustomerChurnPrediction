# 📊 Customer Churn Prediction using Machine Learning

## 📌 Project Overview

Customer churn prediction is a critical business problem that focuses on identifying customers who are likely to discontinue a service or subscription. This project uses Machine Learning techniques to analyze customer behavior and predict whether a customer will leave a service based on historical data.

The objective is to help organizations improve customer retention, reduce revenue loss, and develop effective marketing strategies by identifying at-risk customers before they leave.

---

## 🎯 Objectives

* Predict customer churn using machine learning classification algorithms.
* Analyze customer behavior and historical records.
* Identify key factors influencing customer retention.
* Evaluate model performance using classification metrics.
* Support business decision-making through predictive analytics.

---

## 📊 Dataset Information

The dataset contains customer demographic and account-related information that may influence churn behavior.

### Features Include:

* Credit Score
* Geography
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Credit Card Status
* Active Member Status
* Estimated Salary

### Target Variable

* Exited (0 = Customer Stayed, 1 = Customer Left)

## 🔄 Project Workflow

### 1. Data Collection

Customer data is loaded and inspected for quality and completeness.

### 2. Data Preprocessing

* Missing value handling
* Feature selection
* Label encoding for categorical variables

### 3. Exploratory Data Analysis

* Customer distribution analysis
* Correlation analysis
* Churn visualization

### 4. Model Building

A Random Forest Classifier is trained to identify customers likely to churn.

### 5. Model Evaluation

Performance is measured using:

* Accuracy Score
* Confusion Matrix
* Classification Report

### 6. Prediction

The trained model predicts whether a customer is likely to leave the service.

---

## 📈 Machine Learning Model

### Random Forest Classifier

Random Forest is an ensemble learning algorithm that combines multiple decision trees to improve prediction accuracy and reduce overfitting.

Advantages:

* High accuracy
* Handles large datasets effectively
* Robust against overfitting
* Provides feature importance analysis

---

## 📊 Results

The model successfully classifies customer churn behavior and identifies patterns that influence customer retention.

Key observations:

* Customer age significantly affects churn probability.
* Active members are less likely to leave.
* Product usage influences customer retention.
* Geographic and financial factors impact churn behavior.
